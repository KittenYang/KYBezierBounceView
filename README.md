![](demo.gif)


#Installation

`pod 'KYBezierBounceView', '~> 1.6.1'`


#How to use?

Easy enough! 

Just `#import "BounceView.h"` or use xib/storyboard. 

More details you can See the project demo. 

Enjoy!
